Banco Central do Brasil · SGS

Observatório de Taxas

Série histórica da Meta Selic, do IPCA e do câmbio USD/BRL — dados oficiais do Banco Central, atualizados automaticamente a cada 6 horas, a custo zero.

Sobre o projeto. Este observatório coleta periodicamente as séries do Sistema Gerenciador de Séries Temporais (SGS) do Banco Central do Brasil, registra cada valor como um ponto no histórico e mantém um registro auditável das mudanças ao longo do tempo. Os dados vêm da API pública do BACEN; nada é inserido manualmente. Fonte oficial: www3.bcb.gov.br/sgspub.
