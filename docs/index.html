Observatório de Taxas

Indicadores oficiais do Banco Central do Brasil, versionados no Git e atualizados automaticamente a cada 6 horas — sem servidor e a custo zero.

Sobre o projeto. Este observatório coleta periodicamente as séries do Sistema Gerenciador de Séries Temporais (SGS) do Banco Central, registra cada valor como um ponto no histórico e mantém um registro auditável das mudanças ao longo do tempo. Os dados vêm da API pública do BACEN; nenhuma informação é inserida manualmente. Fonte oficial: www3.bcb.gov.br/sgspub.
